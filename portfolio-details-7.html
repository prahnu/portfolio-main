Portfolio Details

An engaging compass-style diagram that lets users explore different stages of onboarding and learning by clicking on segments, with dynamic content and arrow movement guiding the journey.

Project information

  • Category: Interactive Graphic design
  • Client: EY
  • Tools used: Adobe Illustrator, HTML, CSS, JavaScript

Graphic Design

This application features an engaging, compass-style circular diagram that visually maps out the key stages of an onboarding and continuous learning journey. The initial visual design was crafted in Adobe Illustrator for precise vector graphics and polished aesthetics. In the next phase, Figma was used for prototyping user flows and interactivity, ensuring a seamless experience before development.

  • Dynamic Circular Navigation: The diagram is divided into six labeled segments, each representing a distinct stage or component:
    1. Mastering story creation
    2. Meet the W&CS team
    3. Program admin and operations
    4. Onboarding materials
    5. Using the playbook
    6. Supportive learning
  • Interactive Arrow & Content Reveal: At the center, a red arrow points to the currently selected segment, while additional grey arrows indicate other directions. When users click on any segment, the arrow dynamically rotates to highlight their selection and the relevant content for that stage is displayed.
  • Directional Guidance: The outer edge of the circle features labels such as “Get On Board” and “Continue Learning and Development,” with arrows guiding users through the process.
  • Technology Stack: The final application was developed as a single-page web component using HTML, SVG, CSS, and JavaScript, enabling smooth animations and responsive interactivity.
  • User Experience: The design blends engaging visual storytelling with intuitive navigation, making complex onboarding or learning journeys easy to explore and understand. Users can interact with the diagram to access information relevant to each stage, enhancing both engagement and comprehension.

This project demonstrates the integration of advanced design tools and modern web technologies to deliver a highly interactive and visually compelling user experience for onboarding and continuous learning.
